FIGlet Font Installer (figinstall.exe)

NAME

FIGinstall - FIGfont and FIGlet control file installer.

SYNOPSIS

DESCRIPTION

This program installs new FIGfonts, control files, and TOIlet fonts to the default font directory. It can install not only local fonts, but also these in remote font base.

FIGinstall can also call chkfont.exe to check the install target (only available for FIGfont 2.0 format fonts).

OPTION

  • -L, -l prefix

List FIGfonts, control files and TOIlet fonts in remote font base. With -L option FIGinstall will list all objects, while with -l option only these match the given prefix.

  • -r

Enable restrictive mode. FIGinstall will call chkfont.exe to check the format of the install target. If the install target is a control file or a TOIlet font, this option will be disabled.

  • -R

Get font from remote. FIGinstall will search and try to get this object from remote font base. Use -L option or visit https://github.com/Ace-Radom/FIGfont_base to see which are available.

If you don't use -R option, you should give the local path of the install target at the end of the command.

BUGS

FIGinstall doesn't support proxy. That means that if your computer cannot do requests to GitHub API, FIGinstall will report errors such as "Cannot resolve host".

If you are using a VPN in order to connect to GitHub, please make sure that TUN is enabled.
