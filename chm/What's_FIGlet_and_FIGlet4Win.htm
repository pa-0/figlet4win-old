What's FIGlet and FIGlet4Win

  • What is FIGlet?

FIGlet is a program for making large letters out of ordinary text. It can create characters in many different styles and can kern and "smush" these characters together in various ways.

  • Where can I get FIGlet from?

From the official FIGlet website or from the FIGlet FTP site .

  • What is the difference between FIGlet and FIGlet4Win?

FIGlet is a really old project which started in 1991, and its last "update" was in 2015. Truly the original FIGlet is stable enough and doesn't need much new features any more, but it doesn't change that the FIGlet source code cannot be compiled under modern Windows systems (like Windows 10), and lots of the codes are Unix-like-based.

FIGlet4Win is a project forked from the original FIGlet, to let it operate on mordern Windows systems just like it should be.

  • What has been changed in FIGlet4Win?

Not quite much.

The only user-related change is: the default font directory has been set by default to the fonts directory under the FIGlet4Win install directory (was /usr/local/share/figlet), and relative default font directory will always be converted to a absolute path based on the install directory. That means: using a relative default font directory and make FIGlet4Win to read multiple default font directories under different pathes will not work.

This is mainly because the differences of file path formats between Windows and Unix-like systems.

FIGlet4Win also brings a new component Figinstall, which installs font files to FIGlet default font directory.

  • Can FIGlet4Win be compiled under other operating systems?

It should, but that's not the goal of FIGlet4Win. Therefore compatibility with other platforms is not considered.

  • Where can I get more fonts for FIGlet?

From the FTP site or from the FIGlet font database. There is also a font database on GitHub (https://github.com/cmatsuoka/figlet-fonts) created by Claudio Matsuoka, a developer of the FIGlet project.

There are currently over 400 fonts available for download, hope you will find your one and enjoy!

  • Where does the name FIGlet come from?

It means "Frank, Ian and Glen's letters".
